# NLP-SQL
Natural language query system

Quick Setup

1. Create and activate virtual Environment :
   $ virtualenv
   
   $ source /bin/activate
 
2. Download required packages :
   $ pip install -r requirements.txt 
  
3. Download NLTK packages : 
   $ python 
   >>> import nltk
   
   >>> nltk.download()
  
4. Run Server :
   $ python nlp-sql.py
